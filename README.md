# Hydrogen

Hydrogen is a built-from-scratch alternative UI for the Spotify Car Thing. Currently, it's a work in progress.

## Roadmap

-   [ ] Simple UI
    -   [ ] Now Playing
        -   [ ] Playback controls
        -   [ ] Artist/Album/Track information
-   [ ] Desktop Agent
-   [ ] Installer
-   [ ] Mobile Agent
-   [ ] Advanced UI
    -   [ ] Connect to music service APIs
